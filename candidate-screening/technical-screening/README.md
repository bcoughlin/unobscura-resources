Platforms used by tech companies to assess software engineering candidates through coding challenges, system design exercises, and technical interviews.

## Common Challenges

Most technical screening platforms share these issues:

**Time Pressure:**
- Favors speed over thoughtfulness
- Discriminates against methodical problem-solvers
- Processing speed ≠ code quality

**Live Coding:**
- Performance anxiety under observation
- "Think aloud" requirement difficult for many
- Doesn't reflect real development work

**Algorithm Focus:**
- Whiteboard problems ≠ actual job skills
- Pattern recognition over practical knowledge
- Excludes self-taught developers

## General Accommodation Strategies

**Request:**
- Untimed or extended time option
- Take-home project instead of live coding
- Written explanation instead of verbal
- Ability to research/use documentation

**Propose Alternatives:**
- GitHub portfolio review
- Pair programming session
- Real-world debugging exercise
- Code review discussion of existing work

**During Assessment:**
- Comment your code thoroughly
- Explain your thought process in writing
- Ask clarifying questions
- Demonstrate problem-solving approach even if incomplete
